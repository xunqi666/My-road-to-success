# My-road-to-success
**Record my struggle journey.**

在我大一的时候，我励志创业赚很多钱，但是我是一个极易被环境所影响的人，下学期的时候，我几乎忘记了我这个目标，每天刷短视频8~9个小时，在暑假的时候又染上了游戏，不分昼夜的打游戏。到了大二上学期末的时候才把游戏给戒了，这期间一直浑浑噩噩。直到最近我才开始慢慢开始步入正轨。我觉得记笔记并且把笔记开源可以督促自己不要堕落，帮助和警戒后人，还有我成功后可以来看我的来时路，所以我就创建了这个仓库把我所有的笔记都开源，包括我的思考沉淀、学习点滴、阅读的书籍、每天的行为、等等。

最后，希望大家在读看我笔记时可以给我提供建议、也希望给一些人以启示。每周日更新。
